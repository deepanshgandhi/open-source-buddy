# open-source-buddy
Find open-source issues that match your skills.

## Setup

1. **Prerequisites**: Python 3.12+

2. **Install dependencies**:
   ```bash
   uv venv
   source .venv/bin/activate
   uv pip install -e ".[dev]"
   ```

3. **Configure environment**:
   ```bash
   cp .env.template .env
   # Edit .env with your tokens
   ```
